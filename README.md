# PostgreLaravel

フロントエンドにNext.js + TypeScript、バックエンドにLaravel + PostgreSQLを使用して作成した、商品管理、販売が出来るサンプルアプリケーション。
Vercel / Railwayにデプロイしていて、ユーザー登録・ログイン、商品閲覧、カート管理、画像アップロードなどが可能です。

##　技術スタック

### フロントエンド

- Next.js (App Router)
- TypeScript
- Axios
- react-dropzone
- Tailwind / CSS Modules
- Jest + @testing-library/react

### バックエンド

- Laravel
